Projects

  • Open Radio Access Network (O-RAN) Systems at TWiST Lab
    Actively contributed to the design and implementation of an O-RAN–compliant radio access network, focusing on modularity, flexibility, and interoperability across disaggregated RAN components. Worked with open interfaces and software-defined radio (SDR) platforms, and evaluated system performance under realistic wireless conditions to support next-generation (6G-ready) network architectures.
  • Space–Ground Integrated Communication Systems
    Implemented communication protocols for space–ground integrated systems, addressing challenges related to heterogeneous link characteristics, latency, and reliability across satellite and terrestrial networks.
  • Machine Learning for Real-Time Image Enhancement
    Developed and implemented machine learning algorithms for image enhancement and green-screen removal, optimizing pipelines for real-time processing and low-latency inference.
  • IoT-Based Intrusion Detection System (IDS)
    Led the development of an intrusion detection system for smart homes using Raspberry Pi and IoT sensors, enabling real-time monitoring and detection of unauthorized activities through edge-based processing.
  • Image Processing for Crack Localization
    Developed an image processing pipeline using edge detection and texture analysis techniques to achieve accurate crack localization for structural inspection applications.
